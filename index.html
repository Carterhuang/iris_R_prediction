Iris Prediction RESTful APT
Author: Carter Huang
Email: hh548@cornell.edu

This RESTful API provides ways for users to make predictions based on R iris dataset.

Examples:

To predict Sepal.Length of a data sample:
curl -i -H "Content-Type: pplication/json" -X POST -d '{ "Sepal.Width" : 5, "Petal.Length" : 5, "Petal.Width" : 5, "Species" : "virginica"}' http://54.94.239.97/iris-predict/api/v1.0/predict

To predict Sepal.Width of a data sample:
curl -i -H "Content-Type: pplication/json" -X POST -d '{ "Sepal.Length" : 5, "Petal.Length" : 5, "Petal.Width" : 5, "Species" : "virginica"}' http://54.94.239.97/iris-predict/api/v1.0/predict

To predict Petal.Length of a data sample:
curl -i -H "Content-Type: pplication/json" -X POST -d '{ "Sepal.Length" : 5, "Sepal.Width" : 5, "Petal.Width" : 5, "Species" : "virginica"}' http://54.94.239.97/iris-predict/api/v1.0/predict

To predict Petal.Width of a data sample:
curl -i -H "Content-Type: pplication/json" -X POST -d '{ "Sepal.Length" : 5, "Sepal.Width" : 5, "Petal.Length" : 5, "Species" : "virginica"}' http://54.94.239.97/iris-predict/api/v1.0/predict

To predict Species of a data sample: curl -i -H "Content-Type: pplication/json" -X POST -d '{ "Sepal.Length" : 5, "Sepal.Width" : 5, "Petal.Length" : 5, "Petal.Width" : 5}' http://54.94.239.97/iris-predict/api/v1.0/predict

Implementation:

The API is supported by a backend server. The server consists of two parts, a Python HTTP server and R computation models.

The Python serving code includes implementation of a main server thread ("server.py") and a R client that connects to R models.

As for R, the first model is a weighted k-nearest-neighbor model that predicts the "species" of a data sample based on "Sepal.Length", "Sepal.Width" and so on.

The second model is a linear regression model that predicts non-species attributes such as "Sepal.Length", "Sepal.Width", so on and so forth. One thing to notice is that the regression is implemented on the sub-datasets with the same species.
